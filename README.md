# 2. 提交代码
## 查看修改的文件
```bash
git status
# 如果有人提交新代码，需要将新代码更新到本地
git pull
```
## 查看修改的内容
```bash
git diff 
```
## 提交代码
```bash
# 将所有修改提交到缓冲区
git add .
# 添加提交注释
git commit -a 
# push 到远程仓库
git push
```
# 3. 查看分支
```bash
# 查看所有分支
git branch -a
# 查看当前分支
git branch
```
# 4. 第一次拉取带代码,切换到新分支
```bash
git checkout -b new_branch_name remote_branch_name
# 切换到新分支
git switch branch_name
```

